# STEP1 現状分析ヒアリングシート（記入用）

このシートを埋めると、STEP2以降の診断精度が上がります。埋めた保有一覧は
`data/holdings.csv` に転記してツールで分析できます。

> 記入内容は個人の資産情報です。共有リポジトリにコミットしないでください
> （`data/holdings.csv` は必要に応じて `.gitignore` 対象にすることを推奨）。

## D. スクリーンショットからの書き起こし手順

楽天証券アプリの「保有商品一覧」を開き、各行から次を読み取って上表 C に転記します。

1. 銘柄名（例: eMAXIS Slim 米国株式(S&P500)）
2. 保有口座（NISA/特定）
3. 取得金額（または平均取得単価×口数）
4. 評価額（時価評価額）
5. 積立設定額（「積立設定」画面の毎月金額）

`data/fund_reference.csv` に載っていない銘柄は、同ファイルに1行追加すれば自動分類されます。
