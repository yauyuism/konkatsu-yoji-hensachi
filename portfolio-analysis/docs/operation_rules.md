# STEP5 運用ルール（資産形成のための行動規範）

目的は **短期売買ではなく長期の資産形成**。相場を当てにいくのではなく、
「決めたルールを淡々と実行する」ことで、感情による失敗を構造的に減らします。

> 教育目的のフレームです。投資助言ではありません。金額・比率はご自身の状況に合わせて調整してください。

---

## 1. 毎月の積立ルール

- **自動積立で "先取り"**：給与が入ったら使う前に積み立てる（手動判断を挟まない）。
- **NISA枠を優先**：つみたて投資枠 → 成長投資枠 → 特定口座 の順で埋める（→ §5）。
- **コア・サテライト**：
  - コア（70〜90%）= 全世界株（オルカン）または S&P500 の低コストインデックス。
  - サテライト（0〜30%）= NASDAQ100・新興国・高配当など、自分が納得できる範囲だけ。
- **金額は固定**：相場が上でも下でも同額を続ける（ドルコスト平均）。増額は収入が増えたときだけ。

## 2. 暴落時の追加投資ルール（狼狽の逆をやる仕組み）

前提: **生活防衛資金（生活費6〜12か月）とは別建て**の「暴落用待機資金」を用意しておく。

| トリガー（高値からの下落率） | アクション | 待機資金からの投入目安 |
| --- | --- | --- |
| −10% | 通常積立を継続（何もしない） | 0% |
| −20% | 追加投資 第1弾 | 待機資金の 1/3 |
| −30% | 追加投資 第2弾 | 待機資金の 1/3 |
| −40%以上 | 追加投資 第3弾 | 残り全額 |

- 一括で入れず **分割**。底は誰にも当てられない前提で「下がるほど買い増す」を機械的に。
- 追加投資してもコア資産（オルカン/S&P500）に入れる。個別の話題株には向かわない。
- ニュースやSNSで判断しない。**価格トリガーだけ**を見る。

## 3. 利益確定ルール（資産形成では原則"売らない"）

- 積立期（資産形成フェーズ）は **原則として利確しない**。複利を止めないため。
- 売るのは次の3つに限定：
  1. **ライフイベントでの現金化**（住宅・教育・事業資金など、使う目的が確定したとき）。
  2. **リバランス**（§4。増えすぎたクラスを目標比率まで戻すための部分売却）。
  3. **出口フェーズ**（取り崩し期）に、年3.5〜4%程度の定率/定額で計画的に取り崩す。
- 「上がったから」「怖いから」で売らない。目標比率とライフプランだけが売却理由。

## 4. リバランスルール

- **頻度**：年1回（例: 誕生月や年末）＋大きく動いたときのスポット確認。
- **閾値**：目標比率から **±5%** を超えたクラスだけ調整（小さなズレは放置してコスト削減）。
- **売らずに直す優先**：新規積立や暴落用資金の投入先を、不足クラスに寄せて調整（ノーセル・リバランス）。
  課税口座での売却は税コストが出るため最後の手段。
- ツールの `rebalance_plan.csv` が「増やす/減らす/許容帯」を自動判定します。

## 5. NISA優先活用ルール

- **埋める順番**：つみたて投資枠 → 成長投資枠 → 特定口座。
- **コアはNISAに**：長期保有の中核（オルカン/S&P500）を非課税枠に置き、複利を非課税で伸ばす。
- **特定口座の役割**：NISA枠を超える分、値動きの大きいサテライト、近い将来使う予定の資金。
- **枠の再利用**：NISAは売却すると翌年に非課税枠が復活（簿価ベース）。ただし復活は翌年・年間枠の範囲内。
- **夫婦・家族**：それぞれの名義でNISA枠を使えるため、世帯単位で非課税枠を最大化。

## 感情で売買しないためのチェックリスト（売買ボタンを押す前に）

**「売りたい」と思ったとき**
- [ ] その売却は §3 の3条件（ライフイベント / リバランス / 出口フェーズ）のどれかに当てはまる？
- [ ] 当てはまらないなら、それは「価格の下落」か「不安」が理由では？
- [ ] 5年後・10年後の自分は、今日この売却を評価する？
- [ ] 売った資金の "次の置き場所" は決まっている？（決まっていないなら売らない）

**「買い増したい／もっと攻めたい」と思ったとき**
- [ ] それは §2 の価格トリガーに沿った行動？ それとも高値での高揚感？
- [ ] 生活防衛資金には手を付けていない？
- [ ] コア資産の範囲内？ 話題株・レバレッジ・個別テーマに逸れていない？

**共通（毎回）**
- [ ] SNS・ニュース・含み損益の数字を見て、衝動的に動こうとしていないか。
- [ ] 今日の決定は「事前に決めたルール」に基づくか、「今の気分」に基づくか。
- [ ] 迷ったら **何もしない**（積立継続がデフォルト）。24時間置いてから再検討する。

> 原則: **ルールが行動を決める。気分は行動を決めない。**
> ルールを変えたくなったら、相場が動いていない平常時に、根拠を書いてから変える。
